Detailed Description

Stereo camera base class.

An abstract class to process stereo images from arbitrary cameras. This class should not be used directly, instead you should subclass it (e.g. StereoCameraDeimos()). The bare minimum is to implment a capture function that will capture a stereo pair from your camera.
